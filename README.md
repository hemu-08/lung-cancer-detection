# Lung Cancer Early Detection Web App

AI-powered early detection system for lung cancer in non-smokers using clinical symptoms.

## Features
- Predicts lung cancer risk based on 8 clinical symptoms
- 84% accuracy using Random Forest classification
- Real-time risk probability score
- Clean and colorful web interface

## Tech Stack
- Python
- Scikit-learn (Random Forest)
- Flask
- HTML/CSS

## How to Run
1. Clone the repository
2. Install dependencies: `pip install flask scikit-learn pandas`
3. Run: `python app.py`
4. Open browser: `http://127.0.0.1:5000`
